# World Cup Predictor - Setup Guide

## 🚀 Quick Start (Recommended)

Run the interactive setup script:

```powershell
cd worldcup-predictor
.\scripts\prompt-and-setup.ps1
```

This will prompt you for all required API keys and configure everything automatically.

## 📋 Prerequisites

### 1. Vercel Token
- Go to https://vercel.com/account/tokens
- Click "Create"
- Name it "worldcup-predictor-sync"
- Copy the token

### 2. Football-Data.org API Key (Free)
- Go to https://www.football-data.org/client/register
- Register for a free account
- Copy your API key

### 3. Supabase Service Role Key
- Go to https://supabase.com/dashboard
- Select your project (bpyqxdgpmrybrqjerkfw)
- Go to Settings → API
- Copy the "service_role" key (not the anon key)

## 🔧 Manual Setup

If you prefer to set up manually:

### Step 1: Set Environment Variables

```powershell
$env:VERCEL_TOKEN = "your-vercel-token"
$env:FOOTBALL_DATA_KEY = "your-football-data-key"
$env:SUPABASE_SERVICE_ROLE_KEY = "your-supabase-service-role-key"
```

### Step 2: Run Full Setup

```powershell
.\scripts\full-setup.ps1
```

### Step 3: Enable Supabase Realtime

1. Go to https://supabase.com/dashboard
2. Select your project
3. Go to Database → Replication
4. Enable Realtime for the "matches" table

## 🔄 Automated Pipeline

The sync pipeline runs every 2 minutes:

1. **Data Sync**: Fetches live match data from API-Football or Football-Data.org
2. **Auto-Review**: Generates reviews for finished matches
3. **Weight Update**: Optimizes model weights every 10 new reviews
4. **Frontend Refresh**: Supabase Realtime updates the UI

## 🛠️ Troubleshooting

### Pipeline Not Running
1. Check GitHub Actions: https://github.com/chundongtou/worldcup-predictor/actions
2. Verify secrets are set: `gh secret list -R chundongtou/worldcup-predictor`
3. Check Vercel cron: https://vercel.com/dashboard → Project → Settings → Crons

### API Key Issues
1. Verify Football-Data.org key: https://www.football-data.org/documentation/api
2. Check Supabase keys: https://supabase.com/dashboard → Settings → API

### Deployment Issues
1. Check Vercel deployment: https://vercel.com/dashboard
2. Verify environment variables are set
3. Redeploy if needed
